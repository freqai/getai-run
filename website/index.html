常见问题

什么是 getai.run，它提供什么？

getai.run 是一个 AI API 中转站，为 OpenAI、Claude、Gemini、Codex 等常见模型和客户端提供统一接入入口。

getai.run 上有哪些 AI 模型可用？

可用模型会随上游与套餐变化调整，你可以通过文档或 /v1/models 查看当前列表。

与其他 API 提供商相比，定价如何？

定价以透明和可预测为目标，支持按 token / 按次查看消耗，并提供适合组合使用的阶梯方案。

如何将 getai.run API 集成到我的应用程序中？

通常只需要把客户端的 base_url 改为 https://api.getai.run，并替换为你的 getai.run API Key。

相比直接使用模型提供商，我为什么要选择 getai.run？

它可以减少多平台账号、密钥、协议和账单管理成本，也更适合团队统一出口和自动化场景。

我的 API Key 安全吗？

API Key 用于请求认证。请像管理密码一样保管密钥，避免放入公开仓库、前端代码或日志中。
